# AI Project Template

AI-first engineering project template.

## Structure

- `product/` — features, PRDs, user stories
- `project/` — tasks and workflow
- `docs/` — architecture and sequence
- `src/` — implementation
- `tests/` — validation

## Workflow

1. Define feature (`product/features/`)
2. Create tasks (`project/backlog/`)
3. Follow `docs/TASK_SEQUENCE.md`
4. Implement via AI

poetry install
